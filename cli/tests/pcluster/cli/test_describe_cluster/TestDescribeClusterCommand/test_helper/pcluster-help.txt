usage: pcluster describe-cluster [-h] --cluster-name CLUSTER_NAME
                                 [--region REGION] [--debug]

Get detailed information about an existing cluster.

optional arguments:
  -h, --help            show this help message and exit
  --cluster-name CLUSTER_NAME
                        Name of the cluster
  --region REGION       AWS Region. Defaults to the region the API is deployed
                        to.
  --debug               Turn on debug logging.
