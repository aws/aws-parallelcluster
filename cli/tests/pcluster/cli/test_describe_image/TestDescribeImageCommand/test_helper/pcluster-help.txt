usage: pcluster describe-image [-h] --image-id IMAGE_ID [--region REGION]
                               [--debug] [--query QUERY]

Get detailed information about an existing image.

optional arguments:
  -h, --help           show this help message and exit
  --image-id IMAGE_ID  Id of the image.
  --region REGION      AWS Region that the operation corresponds to.
  --debug              Turn on debug logging.
  --query QUERY        JMESpath query to perform on output.
